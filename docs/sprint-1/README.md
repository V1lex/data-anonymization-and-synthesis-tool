# Документация Sprint 1 (Показ 1)

## Назначение
Этот каталог содержит полный комплект материалов для Показа 1 по MVP проекта SDA.

## Состав
1. `scope.md` - границы MVP и что остается в post-MVP.
2. `architecture.md` - архитектурные слои и правила зависимостей.
3. `api-contracts.md` - API-контракты основных endpoint'ов.
4. `schemas.md` - схемы запросов, ответов и CSV-структур.
5. `templates.md` - каталог шаблонов генерации.
6. `test-plan.md` - тест-план и acceptance criteria для MVP.
7. `ui-flow.md` - пользовательские сценарии и UX-правила.
8. `t009-t010-teamlead-report.md` - отчет по закрытию задач T009/T010.

## Готовность
Материалы в папке достаточны для проведения Показа 1 без дополнительных устных пояснений.
