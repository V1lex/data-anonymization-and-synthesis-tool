## Sprint 3: planned post-MVP

### 1. Suggest rules for anonymization
- Отдельная фича подсказок для `Anonymize`.
- После загрузки CSV система должна уметь рекомендовать метод обработки для колонки.
- Планируемые поля ответа:
  - `suggested_method`
  - `suggestion_confidence`
  - `pii_hint`
- Это расширение не входит в текущий MVP-контур Sprint 1-2 и не требуется для базовой ручной анонимизации.

### 2. Similar data generation
- Загрузка CSV.
- Анализ структуры и базовых статистик.
- Генерация похожего synthetic CSV с теми же колонками.

### 3. Quality improvements
- Более точные эвристики определения PII.
- Более качественная синтетическая генерация и сохранение зависимостей между колонками.
