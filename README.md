# SDA (Synthetic Data Anonymizer)

Учебный проект для двух сценариев:
1. Генерация синтетических CSV по шаблонам.
2. Анонимизация произвольных CSV по правилам на уровне колонок.

## Структура репозитория
1. `src/` — исходный код приложения.
2. `tests/` — тесты.
3. `docs/sprint-1/` — пакет материалов для Показа 1.

## Материалы Показа 1 (`docs/sprint-1/`)
1. `scope.md` — границы MVP и что вне MVP.
2. `architecture.md` — архитектурные слои и правила зависимостей.
3. `api-contracts.md` — API-контракты и формат ошибок.
4. `schemas.md` — схемы запросов, ответов и CSV.
5. `ui-flow.md` — пользовательские сценарии и UX-правила.
6. `test-plan.md` — тест-план, критерии приемки и негативные кейсы.

Этот комплект достаточен, чтобы провести Показ 1 без дополнительных устных пояснений.
